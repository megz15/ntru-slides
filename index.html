Series on Post-Quantum Cryptography

Quantum Threat

  • Quantum computers can factor large numbers in polynomial time
  • Shor's algorithm breaks RSA in hours

Why NTRU?

  • No known quantum algorithm breaks SVP
  • Fast polynomial convolution operations
  • Small keys compared to RSA 2048
  • Already in NIST standardization process

Prerequisites:
Abstract Algebra

What is a Group?

(G, ·): set with a binary operation

  1. Closure: a · b ∈ G for all a, b ∈ G
  2. Associativity: (a · b) · c = a · (b · c)
  3. Identity: ∃ e such that a · e = e · a = a
  4. Inverses: For each a ∈ G, ∃a-1 where a · a-1 = e

If commutative: a · b = b · a → Abelian Group

Example: Arithmetic Modulo 5

3 + 4 ≡ 2 (mod 5)
3 + 4 = 7, and 7 = 1·5 + 2, remainder is 2
  • Closed: Sum always in {0,1,2,3,4}
  • Associative: Addition
  • Identity: 0 (a + 0 ≡ a)
  • Inverse: For 3, inverse is 2 (3+2 ≡ 0 mod 5)

Abelian!

What is a Ring?

(R, +, ·): set with two operations

  1. (R, +) is an abelian group
  2. Multiplication is associative
  3. Distributive law: a·(b+c) = a·b + a·c

Commutative ring: ab = ba for all a,b ∈ R

Polynomial Rings

R[x], where R is a ring.

Consists of all formal sums:

a0 + a1x + a2x2 + ... + anxn where ai ∈ R

Operations:

  • Addition: coefficient-wise
  • Multiplication: polynomial multiplication

Convolution Polynomial Rings

Polynomial quotient rings:

$$R = \frac{\mathbb{Z}[x]}{(x^N - 1)}$$

$$R_q = \frac{(\mathbb{Z}/q\mathbb{Z})[x]}{(x^N - 1)}$$

$x^N = 1$, so exponents reduce modulo $N$

Convolution Product

Multiplication (convolution product) $\star$:

$$a(x) \star b(x) = c(x) \text{ where } c_k = \sum_{i+j \equiv k \pmod{N}} a_i b_{k-i}$$

Example: With $N=5$

$(1-2x+4x^3-x^4) \star (3+4x-2x^2+5x^3+2x^4)$

$= -13 + 20x - 7x^2 + 19x^3 + 5x^4$

Ternary Polynomials

$$T(d_1, d_2) = \{a(x) \in R\}$$

with exactly $d_1$ coefficients equal to $1$, $d_2$ coefficients equal to $-1$, rest $0$

NTRU Key Generation

Public parameters: $(N, p, q, d)$ with $N$ prime, $ \gcd(p,q) = \gcd(N,q) = 1$, $q > (6d+1)p$

Alice's private key:

  • Choose $f(x) \in T(d+1, d)$ and $g(x) \in T(d, d)$
  • Compute $F_q(x) = f(x)^{-1}$ in $R_q$
  • Compute $F_p(x) = f(x)^{-1}$ in $R_p$

Alice's public key:

$$h(x) = F_q(x) \star g(x) \text{ in } R_q$$

NTRU Encryption - Bob

Plaintext:

$m(x) \in R$ with coefficients in $[-\frac{p}{2}, \frac{p}{2}]$

Random ephemeral key $r(x) \in T(d, d)$

Ciphertext:

$$e(x) \equiv p \cdot r(x) \star h(x) + m(x) \pmod{q}$$

NTRU Decryption - Alice

Step 1: Compute intermediate value

$$a(x) \equiv f(x) \star e(x) \pmod{q}$$

Step 2: Center lift $a(x)$ to $R$

Step 3: Recover plaintext

$$b(x) \equiv F_p(x) \star a(x) \pmod{p}$$

Result: $b(x) = m(x)$ when $q > (6d+1)p$

Why Decryption Works

Starting from the ciphertext:

$$a(x) \equiv f(x) \star e(x) \equiv f(x) \star (ph(x) \star r(x) + m(x)) \pmod{q}$$

$$\equiv pf(x) \star F_q(x) \star g(x) \star r(x) + f(x) \star m(x) \pmod{q}$$

$$\equiv pg(x) \star r(x) + f(x) \star m(x) \pmod{q}$$

Why Decryption Works (2)

When $q > (6d+1)p$, coefficients are small enough that:

$$a(x) = pg(x) \star r(x) + f(x) \star m(x) \text{ exactly in } R$$

Therefore:

$$b(x) \equiv F_p(x) \star a(x) \equiv F_p(x) \star f(x) \star m(x) \equiv m(x) \pmod{p}$$

Example: Parameters

$(N, p, q, d) = (7, 3, 41, 2)$

Alice's private key:

$f(x) = x^6 - x^4 + x^3 + x^2 - 1$

$g(x) = x^6 + x^4 - x^2 - x$

$F_p(x) = x^6 + 2x^5+x^3+x^2+x+1$

$F_q(x) = 8x^6 + 26x^5 + 31x^4 + 21x^3 + 40x^2 + 2x + 37$

Public key:

$h(x) = F_q(x) \star g(x)$

$= 20x^6 + 40x^5 + 2x^4 + 38x^3 + 8x^2 + 26x + 30$

Example: Encryption

Bob's message:

$m(x) = -x^5 + x^3 + x^2 - x + 1$

Ephemeral key:

$r(x) = x^6 - x^5 + x - 1$

Ciphertext:

$e(x) = 31x^6 + 19x^5 + 4x^4 + 2x^3 + 40x^2 + 3x + 25$

Example: Decryption

Alice computes:

$f(x) \star e(x) \equiv x^6 + 10x^5 + 33x^4 + 40x^3 + 40x^2 + x + 40 \pmod{41}$

Center lift mod ($q = 41$), coeffs. $\in (-20.5, 20.5]$

$x^6+10x^5-8x^4-x^3-x^2+x-1$

Continuing,

$F_p(x) \star a(x) \equiv -x^5 + x^3 + x^2 - x + 1 \pmod{3}$

Original message!

Breaking NTRU

NTRU Key Recovery Problem

Given: Public key $h(x)$ with random-looking coefficients

Find: Ternary polynomials $f(x), g(x)$ such that

$$f(x) \star h(x) \equiv g(x) \pmod{q}$$

Brute Force Attack

Number of ternary polynomials in $T(d_1, d_2)$:

$$\#T(d_1, d_2) = \binom{N}{d_1}\binom{N-d_1}{d_2} = \frac{N!}{d_1! \cdot d_2! \cdot (N-d_1-d_2)!}$$

Example: $(N, p, q, d) = (251, 3, 257, 83)$

Expected trials: $\frac{\#T(84,83)}{251} \approx 2^{381.6}$

Collision attack (sqrt) $\approx 2^{190.8}$ checks

Still computationally infeasible!
